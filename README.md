## ✨ Features

- ➕ Add items to list
- ✔️ Mark items as Bought / Pending
- 🗑️ Delete items
- 🔍 Filter items (All / Pending / Bought)
- 💾 Data saved using localStorage
- 🔄 Instant UI updates with React state
